Тестирование через Cloudflare Worker

Эта страница создана для проверки того, как Cloudflare Worker подменяет заголовки и эмулирует запросы от имени Googlebot. Мы будем загружать страницу https://cvpdenver.org через iframe, используя настройки Worker-а для подмены User-Agent и Referer.

Детали тестирования

  • Целевая страница: https://cvpdenver.org
  • Источник реферера: https://www.google.com/
  • User-Agent: Googlebot (подменяется через Worker)
  • Проксирование: Происходит через Cloudflare Worker для подмены заголовков.

Цели и задачи тестирования:

Основная цель этого теста — проверить, как Googlebot индексирует контент, подгруженный через iframe с изменёнными заголовками. Это полезно для анализа работы с сайтами, использующими различное поведение для ботов и пользователей.

Эта страница является частью эксперимента по анализу работы поисковых ботов через Cloudflare Worker и не предназначена для публичного использования.
